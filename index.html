Descripción proyectos


• El primer proyecto consistió en desarrollar un juego de ahorcado utilizando el lenguaje de programación C. El objetivo del juego es que el jugador adivine una palabra oculta seleccionando letras una a una. Cada letra incorrecta reduce el número de intentos disponibles, representando partes de una figura de ahorcado en la pantalla.


• El segundo proyecto implicó la creación del clásico juego de Snake, también utilizando C. En este juego, el jugador controla una serpiente que se mueve por la pantalla, creciendo en tamaño cada vez que come comida. El juego termina si la serpiente choca contra las paredes o contra sí misma.


• El tercer proyecto fue el diseño y desarrollo de una página web estática utilizando HTML y CSS. La página web fue diseñada para ser visualmente atractiva y funcional, siguiendo principios de diseño responsivo para asegurar que se vea bien en dispositivos de diferentes tamaños.
